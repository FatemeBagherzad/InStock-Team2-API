# InStock API

This repository contains the backend API for InStock App, which powers the application's core functionality.

## Technologies

<li>Node.js</li>
<li>Express</li>
<li>MySQL</li>
<li>Knex</li>
<li>CORS</li>
<li>dotenv</li>
<li>Nodemon</li>
<li>uniqid</li>
<li>Git, GitHub</li>
<li>Jira</li>
